# Startup Funding EDA Project

Project Objective
To explore and analyze Indian startup funding data and find which industries, cities, and investors have received the highest funding and how trends have changed over time.

---

Steps Performed
1. Imported the dataset (Kaggle source).  
2. Cleaned data â€” removed duplicates, handled missing values, and standardized names.  
3. Performed exploratory data analysis (EDA) using **Python, Pandas, NumPy, and Seaborn**.  
4. Created visualizations like bar charts, line charts, and treemaps to show funding trends.  
5. Found key insights about top industries, top cities, and top investors.  

---

  Key Insights
- E-commerce, Fintech, and HealthTech startups received the highest funding.  
- Bengaluru, Mumbai, and Delhi NCR are the top 3 startup hubs.  
- Most funding came through Series A and B rounds after 2019.
